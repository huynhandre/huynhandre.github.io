Dodecahedron Patterns

Project Description

The pattern subsists of identical dodecahedrons with a certain radius. Starting from a single dodecahedron in the center, every iteration step goes as follows: First, the previously constructed dodecahedrons from the last iteration are duplicated. Then, they are moved away from their current position in orthogonal direction to the twelve faces of the original dodecahedron. The total number of dodecahedrons x for n iterations is:
xn = xn-1 + 12n. The moving distance determines the space between the dodecahedrons in the pattern.

Algorithm

In a for-loop, the input-dodecahedrons are copied and moved in twelve directions. Next, they are saved in a new list in order to serve as the input in the next iteration, until the maximum iteration is reached.

Web App

Use the "distance"-sylider to change the distance between dodecahedrons. With the "iterations"-slider you can choose the desired number of iterations.
